Toto-Cell : a new software to analyze cellular events during video-microscopy

submitted to Plos One

Renaud Léa-Isabelle(1), Asselin Eric(1), Delepoulle Samuel(2) Renaud Christophe(2)

(1) Laboratoire de gynéco-oncologie moléculaire, Université du Québec Trois-Rivières, Trois Rivières, Québec, Canada

(2) Laboratoire d’Informatique, Signal et Image (LISIC), Université du Littoral Côte d’Opale, Calais, 62228, France


Cell is a standalone software application designed to generate SVG visual representations of cellular dynamics through textual analysis of video sequences.

The programm can be download from this repo : https://gitlab.com/crrenaud/cellv1.5

An extended documentation of cell can be downloaded in pdf format

Furthermore, this web-based platform supports convenient upload of text files (.txt) for subsequent analysis.
